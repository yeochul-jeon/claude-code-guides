# CLAUDE.md — claude-code-guides

## 프로젝트 개요

Claude Code 활용 가이드 문서 모음 (한국어). 순수 마크다운 프로젝트로 빌드/의존성 없음.

## 문서 작성 규칙

- 모든 문서는 **한국어**로 작성, 기술 용어는 한/영 병기 허용
- 문서 상단에 메타데이터 테이블 포함 (생성일, 변경일)
- 반드시 목차(TOC) 섹션 포함
- 코드 예시는 언어 태그 명시 (```json, ```bash 등)
- 설명보다 실전 예시 우선

## 디렉토리 구조

- `guides/`: 범용 Claude Code 가이드
- `internal/`: CJ 내부 전용 문서 (Confluence CLI 등)
- `.gitignore`에 민감 정보(VPN 가이드, .env) 등록됨

## 주의사항

- 코드 프로젝트가 아니므로 빌드/테스트 명령 없음
- 문서 수정 시 메타데이터 테이블의 변경일을 오늘 날짜로 업데이트할 것
- 문서 간 내용 중복을 피하고, 관련 내용은 링크로 참조
